# Быстрый старт с Dashboard

## Установка

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Настроить .env
cp .env.example .env
# Отредактировать .env своими ключами
```

## Запуск

```bash
# Запустить бота с dashboard
python -m src.main
```

Dashboard будет доступен по адресу: **http://127.0.0.1:5000**

## Что показывает Dashboard

### Основные метрики
- **Баланс** - текущий баланс в USDT
- **Equity** - баланс + нереализованный PnL
- **Общий PnL** - суммарная прибыль/убыток
- **Винрейт** - процент прибыльных сделок

### Открытые позиции
Таблица с:
- Направление (LONG/SHORT)
- Размер позиции
- Цена входа
- Текущая цена
- PnL в USDT и %

### Статистика
- Всего сделок
- Прибыльных/убыточных
- Средняя прибыль/убыток
- Profit Factor

## Обновление данных

Dashboard автоматически обновляется каждые 2 секунды.

## Режимы работы

### Testnet (рекомендуется)
```env
TRADING_MODE=testnet
```

### Paper Trading (симуляция)
```env
TRADING_MODE=paper
```

### Live Trading (реальные деньги)
```env
TRADING_MODE=live
```

## Структура данных

История сделок сохраняется в `data/trade_history.json`

## Troubleshooting

### Dashboard не открывается
- Проверьте, что порт 5000 свободен
- Проверьте логи в `logs/trading_bot.log`

### Нет данных на dashboard
- Подождите первую закрытую свечу
- Проверьте подключение к Bybit API
- Убедитесь, что бот запущен

### Позиции не отображаются
- Откройте позицию через бота
- Проверьте, что API ключи имеют права на торговлю
