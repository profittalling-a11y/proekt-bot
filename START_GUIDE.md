## 1. Установка

### Шаг 1: Установить Python

Скачайте Python 3.11+ с [python.org](https://www.python.org/downloads/)

Проверьте установку:
```bash
python --version
```

### Шаг 2: Скачать проект

Распакуйте проект в папку, например:
```
C:\Users\desin\OneDrive\Рабочий стол\Proekt Bot
```

### Шаг 3: Создать виртуальное окружение

Откройте командную строку в папке проекта:

```bash
# Создать виртуальное окружение
python -m venv venv

# Активировать (Windows)
venv\Scripts\activate

# Активировать (Linux/Mac)
source venv/bin/activate
```

### Шаг 4: Установить зависимости

```bash
pip install -r requirements.txt
```

## 2. Настройка Bybit

### Вариант A: Demo Trading (Рекомендуется)

#### 1. Зарегистрироваться на Bybit

Перейдите на [bybit.com](https://www.bybit.com) и зарегистрируйтесь.

**Верификация НЕ требуется для Demo!**

#### 2. Включить Demo Trading

1. Войдите в аккаунт Bybit
2. Перейдите в **Derivatives** → **USDT Perpetual**
3. В правом верхнем углу найдите переключатель **"Demo Trading"**
4. Включите Demo режим
5. Вам будет выдано **100,000 USDT** виртуальных денег

#### 3. Создать API ключи для Demo

1. Перейдите в **Account** → **API Management**
2. Нажмите **Create New Key**
3. Выберите **Demo Trading** (важно!)
4. Настройте права:
   - ✅ **Read-Write** (для торговли)
   - ✅ **Contract Trading** (для фьючерсов)
   - ❌ **Withdraw** (отключить для безопасности)
5. Настройте IP whitelist (опционально, но рекомендуется):
   - Узнайте свой IP: [whatismyip.com](https://www.whatismyip.com)
   - Добавьте в whitelist
6. Нажмите **Submit**
7. **Сохраните API Key и Secret** (показываются только один раз!)

#### 4. Настроить .env

Скопируйте `.env.example` в `.env`:
```bash
copy .env.example .env
```

Откройте `.env` и заполните:

```env
# Bybit API (Demo Trading)
BYBIT_API_KEY=ваш_api_key
BYBIT_API_SECRET=ваш_api_secret

# Режим: paper для тестирования без реальных ордеров
# Или используйте testnet если у вас Demo API
TRADING_MODE=testnet

# Автосканирование ликвидных пар
AUTO_SCAN=true
MIN_VOLUME_24H=1000000
MAX_SYMBOLS=10

# Таймфрейм и индикаторы
TIMEFRAME=15
ATR_PERIOD=10
SUPERTREND_MULTIPLIER=3.0

# Риск-менеджмент
FIXED_POSITION_SIZE=1.0
RISK_PER_POSITION=2.0
MAX_POSITIONS=10
MAX_DAILY_LOSS=20.0

# Telegram (опционально)
TELEGRAM_ENABLED=false
TELEGRAM_BOT_TOKEN=
TELEGRAM_CHAT_ID=

# Логирование
LOG_LEVEL=INFO
```

### Вариант B: Paper Trading (Без API)

Если не хотите создавать API ключи, можно использовать **Paper Trading** режим:

```env
TRADING_MODE=paper
BYBIT_API_KEY=dummy_key
BYBIT_API_SECRET=dummy_secret
```

В этом режиме:
- ✅ Получает реальные рыночные данные
- ✅ Генерирует сигналы
- ❌ НЕ открывает реальные ордера
- ✅ Логирует все действия

## 4. Запуск бота

### Запустить веб-сервер

```bash
# Активировать виртуальное окружение
venv\Scripts\activate

# Запустить веб-сервер (бот НЕ запустится автоматически)
python -m src.main
```

### Что должно произойти:

```
==========================================================
Bot initialized. Use dashboard to start trading.
Dashboard: http://127.0.0.1:5000
==========================================================

Auto-scanning markets for liquid pairs...
Found 45 liquid pairs

1. BTCUSDT: $5,234,567,890 volume
2. ETHUSDT: $2,456,789,012 volume
...

Configuration generated:
Symbols: BTCUSDT,ETHUSDT,SOLUSDT,...

Starting dashboard on http://127.0.0.1:5000
```

### Открыть Dashboard и запустить торговлю

1. Откройте браузер: **http://127.0.0.1:5000**
2. Нажмите кнопку **"▶ Запустить"** для начала торговли
3. Для остановки нажмите **"⏹ Остановить"**

**Важно:** Бот начинает торговать только после нажатия кнопки "Запустить" на dashboard!

## 🐛 Решение проблем

### Ошибка: "API Authentication failed"

**Решение:**
1. Проверьте API Key и Secret в `.env`
2. Убедитесь, что ключи для Demo Trading
3. Проверьте IP whitelist на Bybit

### Ошибка: "Insufficient balance"

**Решение:**
1. Проверьте баланс на Demo счете
2. Уменьшите `FIXED_POSITION_SIZE`
3. Уменьшите `MAX_POSITIONS`

### Бот не открывает позиции

**Причины:**
1. Нет сигналов Supertrend (нужно ждать)
2. Риск-лимиты достигнуты
3. Cooldown после убытка активен

**Проверьте логи:**
```
Trading not allowed: Daily loss limit reached
```

### Dashboard не открывается

**Решение:**
1. Проверьте, что бот запущен
2. Откройте http://127.0.0.1:5000
3. Проверьте, что порт 5000 свободен
