## 🎯 Ключевые особенности

### Архитектура

```
ExchangeClient (abstract)
    ├── OKXClient
    ├── BybitClient
    └── BingXClient

ExchangeFactory
    └── create_client(exchange, api_key, api_secret, passphrase)

Config
    ├── get_api_credentials()
    ├── set_api_credentials()
    └── set_trading_params()
```

### API Эндпоинты

**Новые эндпоинты:**
- `GET /api/exchanges` - Список поддерживаемых бирж
- `GET /api/config` - Текущая конфигурация
- `POST /api/config/exchange` - Установка биржи и API ключей
- `POST /api/config/trading` - Установка параметров торговли

**Существующие эндпоинты:**
- `GET /api/status` - Статус бота
- `GET /api/positions` - Открытые позиции
- `GET /api/statistics` - Статистика
- `POST /api/control/start` - Запуск бота
- `POST /api/control/stop` - Остановка бота

### Форматы символов

Бот автоматически нормализует символы для каждой биржи:

| Биржа | Формат | Пример |
|-------|--------|--------|
| OKX | BASE-QUOTE-SWAP | BTC-USDT-SWAP |
| Bybit | BASEQUOTE | BTCUSDT |
| BingX | BASE-QUOTE | BTC-USDT |

### Таймфреймы

Поддерживаемые таймфреймы:
- 5 минут
- 10 минут
- 15 минут (по умолчанию)
- 30 минут
- 60 минут (1 час)
- 120 минут (2 часа)
- 240 минут (4 часа)

### Supertrend Multiplier

Диапазон: 1.0 - 10.0
- **1.0 - 2.0**: Агрессивная торговля
- **2.5 - 3.5**: Сбалансированная (рекомендуется)
- **4.0 - 10.0**: Консервативная

## 📋 Как использовать

### 1. Через дашборд (рекомендуется)

```bash
# Запустите бота
python -m src.main

# Откройте браузер
http://127.0.0.1:5000

# В дашборде:
# 1. Нажмите "⚙️ Настройки"
# 2. Выберите биржу
# 3. Введите API ключи
# 4. Настройте таймфрейм и мультипликатор
# 5. Сохраните настройки
# 6. Запустите бота
```

### 2. Через .env файл

```env
# Выберите биржу
EXCHANGE=okx

# Введите API ключи для выбранной биржи
OKX_API_KEY=your_key
OKX_API_SECRET=your_secret
OKX_PASSPHRASE=your_passphrase

# Настройте параметры
TIMEFRAME=15
SUPERTREND_MULTIPLIER=3.0
TRADING_MODE=testnet
```

## ⚠️ Важные замечания

### API Ключи

1. **OKX требует passphrase** - обязательное поле
2. **Bybit** - только key и secret
3. **BingX** - только key и secret

### Безопасность

1. Используйте **IP Whitelist** на бирже
2. Отключите **вывод средств** для API ключей
3. Начинайте с **demo/testnet** режима
4. Используйте **минимальные суммы** при первом запуске

### Тестирование

1. **Обязательно протестируйте на demo** минимум 1 неделю
2. Проверьте работу на разных таймфреймах
3. Попробуйте разные значения мультипликатора
4. Убедитесь, что stop loss работает корректно

## 🐛 Известные ограничения

1. **Нельзя менять биржу** во время работы бота (нужно остановить)
2. **Нельзя менять параметры** во время работы бота
3. **Один бот = одна биржа** (нельзя торговать на нескольких одновременно)
