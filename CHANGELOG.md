## Как использовать

### Запуск
```bash
python -m src.main
```

### Открыть dashboard
Откройте в браузере: http://127.0.0.1:5000

### Что вы увидите

#### Верхняя панель
- Название бота
- Статус (Работает/Остановлен)

#### Карточки метрик (4 штуки)
1. **Баланс** - текущий баланс + символ
2. **Equity** - баланс с учетом открытых позиций + текущая цена
3. **Общий PnL** - суммарная прибыль/убыток + процент
4. **Винрейт** - процент прибыльных сделок + количество

#### Открытые позиции
Таблица с колонками:
- Направление (зеленый LONG / красный SHORT)
- Размер позиции
- Цена входа
- Текущая цена
- PnL в USDT (зеленый/красный)
- PnL в % (зеленый/красный)

Если позиций нет - показывается "Нет открытых позиций"

#### Статистика
6 блоков с данными:
- Всего сделок
- Прибыльных (зеленым)
- Убыточных (красным)
- Средняя прибыль
- Средний убыток
- Profit Factor

#### Внизу
Время последнего обновления

## Технические детали

### Thread-safe обновление
Dashboard использует Lock для безопасного обновления данных из разных потоков.

### Автоматическое обновление
JavaScript на странице делает запрос к API каждые 2 секунды.

### Расчет PnL для открытых позиций
```python
if direction == 'long':
    pnl = (current_price - entry_price) * size
else:
    pnl = (entry_price - current_price) * size
```

### Цветовая индикация
- Зеленый: прибыль, LONG, работает
- Красный: убыток, SHORT, остановлен
- Серый: нейтральное состояние
