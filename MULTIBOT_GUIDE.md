# Система множественных ботов - Руководство

## 🎯 Обзор

Теперь бот поддерживает **одновременный запуск нескольких ботов** на разных биржах и аккаунтах. Вы можете запустить до 3+ ботов одновременно, каждый со своими настройками.

## ✨ Возможности

- **Одновременная работа**: Запускайте несколько ботов параллельно
- **Разные биржи**: OKX, Bybit, BingX одновременно
- **Независимые настройки**: Каждый бот работает со своими параметрами
- **Централизованное управление**: Управляйте всеми ботами из одного дашборда
- **Мониторинг в реальном времени**: Отслеживайте статус всех ботов

## 🚀 Быстрый старт

### 1. Создайте аккаунты для разных бирж

```
Аккаунт 1: "OKX Main"
- Биржа: OKX
- Таймфрейм: 15 минут
- ST Multiplier: 3.0

Аккаунт 2: "Bybit Scalping"
- Биржа: Bybit
- Таймфрейм: 5 минут
- ST Multiplier: 2.5

Аккаунт 3: "BingX Conservative"
- Биржа: BingX
- Таймфрейм: 30 минут
- ST Multiplier: 4.0
```

### 2. Запустите боты

**Вариант A: Запустить все сразу**
1. Нажмите **▶️ Запустить все**
2. Все боты запустятся одновременно

**Вариант B: Запустить выборочно**
1. Нажмите **👤 Аккаунты**
2. Найдите нужный аккаунт
3. Нажмите **▶️ Запустить** рядом с аккаунтом
4. Повторите для других аккаунтов

### 3. Мониторинг

В секции **🤖 Запущенные боты** отображаются все активные боты:
- Название аккаунта
- Биржа и символ
- Таймфрейм
- Время запуска
- Кнопка остановки

## 🎮 Управление ботами

### Запуск отдельного бота

```bash
# Через дашборд
1. Откройте "👤 Аккаунты"
2. Найдите аккаунт
3. Нажмите "▶️ Запустить"

# Через API
curl -X POST http://localhost:5000/api/control/start \
  -H "Content-Type: application/json" \
  -d '{"account_id": "okx_20260523_103045"}'
```

### Остановка отдельного бота

```bash
# Через дашборд
1. В секции "Запущенные боты"
2. Нажмите "⏹️ Остановить" рядом с ботом

# Через API
curl -X POST http://localhost:5000/api/control/stop \
  -H "Content-Type: application/json" \
  -d '{"account_id": "okx_20260523_103045"}'
```

### Запуск всех ботов

```bash
# Через дашборд
Нажмите "▶️ Запустить все"

# Через API
curl -X POST http://localhost:5000/api/bots/start-all
```

### Остановка всех ботов

```bash
# Через дашборд
Нажмите "⏹️ Остановить все"

# Через API
curl -X POST http://localhost:5000/api/bots/stop-all
```

## 📡 API Эндпоинты

### Управление ботами

```

Body: {"account_id": "account_id"}
Описание: Остановить бота для аккаунта

GET /api/bots/running
Описание: Получить список запущенных ботов

POST /api/bots/start-all
Описание: Запустить боты для всех аккаунтов

POST /api/bots/stop-all
Описание: Остановить все запущенные боты
```

## ⚠️ Важные замечания

### Ограничения

1. **Ресурсы**: Каждый бот потребляет CPU и память
2. **API лимиты**: Учитывайте rate limits бирж
3. **Капитал**: Убедитесь, что на каждом аккаунте достаточно средств
4. **Мониторинг**: Следите за всеми ботами одновременно

### Рекомендации

1. **Начните с 1-2 ботов**: Освойтесь с системой
2. **Тестируйте на demo**: Используйте testnet аккаунты
3. **Разные биржи**: Диверсифицируйте риски
4. **Мониторьте логи**: Проверяйте `logs/trading_bot.log`
5. **Установите лимиты**: Используйте разумные размеры позиций

### Производительность

**Рекомендуемые характеристики системы:**
- CPU: 2+ ядра
- RAM: 4+ GB
- Интернет: Стабильное соединение

**Для 3 ботов:**
- CPU: ~10-20% загрузка
- RAM: ~500-800 MB
- Network: ~1-5 Mbps

## 🐛 Troubleshooting

### Бот не запускается

**Проблема**: Ошибка при запуске бота

**Решение**:
1. Проверьте API ключи аккаунта
2. Убедитесь, что на балансе есть средства
3. Проверьте логи: `logs/trading_bot.log`

### Боты конфликтуют

**Проблема**: Боты мешают друг другу

**Решение**:
1. Используйте разные аккаунты на одной бирже
2. Или используйте разные биржи
3. Не запускайте два бота на одном аккаунте

### Высокая нагрузка

**Проблема**: Система тормозит

**Решение**:
1. Уменьшите количество ботов
2. Увеличьте таймфрейм (меньше запросов)
3. Закройте другие программы
