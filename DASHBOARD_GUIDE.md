# Multi-Exchange Trading Bot - Руководство пользователя

## 🎯 Обзор

Торговый бот с поддержкой нескольких бирж (OKX, Bybit, BingX) и веб-дашбордом для управления.

## ✨ Основные возможности

- **Поддержка 3 бирж**: OKX, Bybit, BingX
- **Веб-дашборд**: Управление через браузер
- **Настройка через UI**: API ключи, таймфрейм, мультипликатор
- **Гибкие таймфреймы**: 5, 10, 15, 30 минут и более
- **Supertrend стратегия**: Настраиваемый мультипликатор (1.0-10.0)
- **Управление рисками**: Фиксированный размер позиции, дневные лимиты
- **Режимы торговли**: Demo/Testnet, Paper, Live

## 🚀 Быстрый старт

### 1. Установка

```bash
# Установите зависимости
pip install -r requirements.txt
```

### 2. Запуск

```bash
# Запустите бота
python -m src.main
```

### 3. Откройте дашборд

Откройте в браузере: **http://127.0.0.1:5000**

## 🎛️ Настройка через дашборд

### Выбор биржи и API ключи

1. Нажмите **⚙️ Настройки** в правом верхнем углу
2. Выберите биржу из списка:
   - **OKX** - требует API Key, Secret и Passphrase
   - **Bybit** - требует API Key и Secret
   - **BingX** - требует API Key и Secret
3. Введите ваши API ключи
4. Нажмите **💾 Сохранить настройки биржи**

### Параметры торговли

1. Выберите **Таймфрейм**:
   - 5 минут
   - 10 минут
   - 15 минут (по умолчанию)
   - 30 минут
   - 1 час
   - 2 часа
   - 4 часа

2. Установите **Supertrend Multiplier** (1.0 - 10.0):
   - Меньше значение = больше сигналов, но больше ложных
   - Больше значение = меньше сигналов, но более надежных
   - Рекомендуется: 2.5 - 3.5

3. Нажмите **💾 Сохранить параметры торговли**

### Запуск бота

1. Убедитесь, что настройки сохранены
2. Нажмите **▶️ Запустить бота**
3. Бот начнет торговлю согласно настройкам

### Остановка бота

Нажмите **⏹️ Остановить бота**

## 📊 Мониторинг

Дашборд показывает в реальном времени:

- **Баланс**: Доступные средства в USDT
- **Equity**: Баланс + нереализованная прибыль
- **Текущая цена**: Цена торгуемого инструмента
- **Открытые позиции**: Таблица с активными сделками
- **Статистика**: Всего сделок, прибыльных, убыточных, Win Rate

## 🔑 Получение API ключей

### OKX

1. Зайдите на [OKX](https://www.okx.com)
2. Перейдите в **API Management**
3. Создайте новый API ключ
4. **Важно**: Сохраните **Passphrase** - он нужен для аутентификации
5. Установите права: **Trade** (торговля), **Read** (чтение)
6. Рекомендуется включить **IP Whitelist**

### Bybit

1. Зайдите на [Bybit](https://www.bybit.com)
2. Перейдите в **API Management**
3. Создайте новый API ключ
4. Установите права: **Contract Trade**, **Position**
5. Для testnet используйте [testnet.bybit.com](https://testnet.bybit.com)

### BingX

1. Зайдите на [BingX](https://bingx.com)
2. Перейдите в **API Management**
3. Создайте новый API ключ
4. Установите права: **Trade**, **Read**

## 🔒 Безопасность

1. **Никогда не делитесь API ключами**
2. **Используйте IP Whitelist** на бирже
3. **Отключите вывод средств** для API ключей
4. **Начинайте с demo/testnet режима**
5. **Используйте минимальные суммы** при первом запуске
6. **Установите разумные лимиты риска**

## 📝 Форматы символов

Каждая биржа использует свой формат:

- **OKX**: `BTC-USDT-SWAP`, `ETH-USDT-SWAP`
- **Bybit**: `BTCUSDT`, `ETHUSDT`
- **BingX**: `BTC-USDT`, `ETH-USDT`

Бот автоматически нормализует символы для выбранной биржи.


## 🎯 Стратегия Supertrend

### Как работает

1. **Индикатор Supertrend** определяет тренд:
   - Зеленая линия = восходящий тренд (LONG)
   - Красная линия = нисходящий тренд (SHORT)

2. **Сигналы**:
   - Смена с красного на зеленый = открыть LONG
   - Смена с зеленого на красный = открыть SHORT

3. **Stop Loss**: Автоматически размещается на уровне Supertrend

### Настройка мультипликатора

- **1.0 - 2.0**: Агрессивная торговля, много сигналов
- **2.5 - 3.5**: Сбалансированная торговля (рекомендуется)
- **4.0 - 10.0**: Консервативная торговля, мало сигналов

## 📈 Таймфреймы

Выбор таймфрейма влияет на стиль торговли:

- **5-10 минут**: Скальпинг, много сделок
- **15-30 минут**: Внутридневная торговля
- **1-4 часа**: Свинг-трейдинг

## ⚠️ Риски и предупреждения

1. **Торговля криптовалютами сопряжена с высоким риском**
2. **Прошлые результаты не гарантируют будущих**
3. **Тестируйте на demo минимум 1 неделю**
4. **Начинайте с минимальных сумм**
5. **Не оставляйте бота без присмотра надолго**
6. **Используйте только те средства, которые можете позволить себе потерять**

## 🐛 Troubleshooting

### Ошибка аутентификации

```
API Error: Invalid sign / Authentication failed
```

**Решение**: Проверьте правильность API ключей и passphrase (для OKX)

### Бот не запускается

**Решение**: 
1. Проверьте, что API ключи введены
2. Убедитесь, что выбрана правильная биржа
3. Проверьте логи в `logs/trading_bot.log`

### Нет сигналов

**Решение**:
1. Попробуйте уменьшить Supertrend Multiplier
2. Измените таймфрейм
3. Проверьте, что рынок активен
