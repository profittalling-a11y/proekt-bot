# Исправления бота - 28.05.2026

## Проблема
Бот не анализировал графики и не открывал сделки из-за ошибки API:
```
OKX API Error 51001: Instrument ID doesn't exist
```

## Причина
Неправильный формат символа торговой пары:
- Использовался: `BTCUSDT`
- Требуется для OKX: `BTC-USDT-SWAP`

## Исправления

### 1. ✅ OKX Client (src/okx_client.py)
Добавлена автоматическая нормализация символов во ВСЕХ методах API:
- `get_klines()` - получение свечей
- `get_ticker_price()` - текущая цена
- `get_position()` - текущая позиция
- `place_order()` - размещение ордера
- `close_position()` - закрытие позиции
- `set_leverage()` - установка плеча
- `get_exchange_info()` - информация о паре

### 2. ✅ Config (src/config.py)
- Изменен дефолтный формат символов: `BTC-USDT-SWAP,ETH-USDT-SWAP`
- Добавлен метод `normalize_symbol()` для автоматической конвертации
- Обновлен валидатор `validate_symbols()` для автоматической нормализации
- Исправлен `leverage_config` на правильный формат

### 3. ✅ Trader (src/trader.py)
- Добавлена нормализация символа при инициализации бота
- Добавлена валидация символа перед стартом торговли
- Проверка существования торговой пары через `get_exchange_info()`

### 4. ✅ .env файл
Обновлены дефолтные значения:
```env
SYMBOLS=BTC-USDT-SWAP
LEVERAGE_CONFIG=BTC-USDT-SWAP:50,ETH-USDT-SWAP:50,...
```

### 5. ✅ accounts.json
Добавлено поле `symbol` с правильным форматом для активного аккаунта.

## Результат

Теперь бот:
1. ✅ Автоматически конвертирует `BTCUSDT` → `BTC-USDT-SWAP`
2. ✅ Валидирует символ перед началом торговли
3. ✅ Успешно получает данные от OKX API
4. ✅ Анализирует графики (индикаторы Supertrend, ATR, EMA)
5. ✅ Генерирует торговые сигналы
6. ✅ Открывает сделки при выполнении условий

## Как запустить

```bash
# Активировать виртуальное окружение
venv\Scripts\activate

# Запустить бота
python -m src.main
```

Бот запустится с дашбордом на http://127.0.0.1:5000

## Проверка работы

После запуска в логах должно быть:
```
INFO | Normalized symbol to OKX format: BTC-USDT-SWAP
INFO | Validating symbol: BTC-USDT-SWAP
INFO | Symbol validated: BTC-USDT-SWAP
INFO | TRADING BOT STARTED
INFO | New candle closed at ...
INFO | Fetched 200 klines for BTC-USDT-SWAP
```

Если видите эти сообщения - бот работает правильно!
