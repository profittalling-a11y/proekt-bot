# Настройка Telegram уведомлений

## 📱 Как получить Telegram Bot Token и Chat ID

### Шаг 1: Создать бота

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Введите имя бота (например: `My Trading Bot`)
4. Введите username бота (должен заканчиваться на `bot`, например: `my_trading_bot`)
5. BotFather отправит вам **Bot Token** - скопируйте его

Пример токена: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`

### Шаг 2: Получить Chat ID

**Вариант 1: Через @userinfobot**
1. Найдите в Telegram **@userinfobot**
2. Отправьте ему любое сообщение
3. Бот ответит вашим Chat ID

**Вариант 2: Через API**
1. Отправьте сообщение вашему боту (тому, что создали в Шаге 1)
2. Откройте в браузере:
   ```
   https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
   ```
3. Найдите `"chat":{"id":123456789}` - это ваш Chat ID

### Шаг 3: Настроить .env

Добавьте в файл `.env`:

```env
TELEGRAM_ENABLED=true
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_CHAT_ID=123456789
```

### Шаг 4: Проверить подключение

Запустите бота - при старте придет сообщение:
```
🚀 Бот запущен

Торговый бот успешно запущен и готов к работе.
```

## 🔧 Дополнительные настройки

### Отключить уведомления

В `.env`:
```env
TELEGRAM_ENABLED=false
```

### Групповой чат

Можно отправлять уведомления в группу:
1. Добавьте бота в группу
2. Сделайте его администратором
3. Получите Chat ID группы (будет отрицательным, например: `-1001234567890`)
4. Укажите в `.env`:
   ```env
   TELEGRAM_CHAT_ID=-1001234567890
   ```

### Несколько получателей

К сожалению, напрямую нельзя. Но можно:
1. Создать группу
2. Добавить всех нужных людей
3. Использовать Chat ID группы

## 🎨 Форматирование сообщений

Сообщения используют HTML форматирование:
- **Жирный текст**: `<b>текст</b>`
- *Курсив*: `<i>текст</i>`
- Эмодзи для быстрого распознавания:
  - 🟢 LONG позиция
  - 🔴 SHORT позиция
  - ✅ Прибыль
  - ❌ Убыток
  - 🚀 Запуск
  - 🛑 Остановка
  - ⚠️ Ошибка

## 🔒 Безопасность

### Важно!

1. **Не публикуйте Bot Token** - это ключ к вашему боту
2. **Не делитесь Chat ID** - иначе кто-то может отправлять вам сообщения
3. **Храните .env в .gitignore** - не коммитьте в Git
4. **Используйте отдельного бота** для каждого проекта

### Если токен скомпрометирован

1. Откройте @BotFather
2. Отправьте `/mybots`
3. Выберите бота
4. Нажмите `API Token`
5. Нажмите `Revoke current token`
6. Получите новый токен

## 🧪 Тестирование

### Проверить подключение

Можно добавить тестовую команду в код:

```python
from src.telegram_notifier import TelegramNotifier

notifier = TelegramNotifier(
    bot_token="YOUR_TOKEN",
    chat_id="YOUR_CHAT_ID",
    enabled=True
)

# Проверка
if notifier.test_connection():
    print("✅ Telegram подключен!")
else:
    print("❌ Ошибка подключения")
```
