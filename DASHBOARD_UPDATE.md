## Как работает

### Запуск бота для пары:
1. Выберите аккаунт (кнопка "👤 Аккаунты")
2. В любом из 10 блоков:
   - Выберите торговую пару
   - Настройте таймфрейм
   - Настройте Supertrend Multiplier
   - Нажмите "▶️ Старт"

### Остановка бота:
- Нажмите "⏹️ Стоп" в нужном блоке

### Мониторинг:
- **Активных ботов**: показывает сколько ботов работает (X / 10)
- **Баланс**: текущий баланс USDT
- **Equity**: баланс + нереализованный PnL
- **Общий PnL**: суммарный PnL по всем позициям

## API эндпоинты (требуется реализация)

Дашборд использует следующие API:
- `POST /api/multibot/start` - запуск бота для пары
- `POST /api/multibot/stop` - остановка бота
- `GET /api/all` - получение всех данных дашборда
- `GET /api/accounts` - список аккаунтов
- `POST /api/accounts/{id}/activate` - активация аккаунта
