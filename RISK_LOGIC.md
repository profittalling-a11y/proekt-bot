# Правильная логика Risk Management

## ✅ Исправлено

### 1. Фиксированный размер позиции с масштабированием

**Логика:**
- Базовый размер: **1 USDT** (фиксированный)
- Масштабирование: каждые **100 USDT** баланса добавляют **1x** множитель

**Примеры:**
```
Баланс 100 USDT  → Позиция 1 USDT
Баланс 150 USDT  → Позиция 1 USDT (множитель 1)
Баланс 200 USDT  → Позиция 2 USDT (множитель 2)
Баланс 250 USDT  → Позиция 2 USDT (множитель 2)
Баланс 350 USDT  → Позиция 3 USDT (множитель 3)
Баланс 1000 USDT → Позиция 10 USDT (множитель 10)
```

**Код:**
```python
balance_multiplier = int(balance / 100)
if balance_multiplier < 1:
    balance_multiplier = 1
position_size_usdt = fixed_position_size * balance_multiplier
```

### 2. Плечо по монете (ручная настройка)

**Конфигурация в .env:**
```env
LEVERAGE_CONFIG=BTCUSDT:50,ETHUSDT:50,SOLUSDT:25,BNBUSDT:25,XRPUSDT:20,ADAUSDT:20,DOGEUSDT:20,MATICUSDT:25,DOTUSDT:25,AVAXUSDT:25
```

**Логика:**
- Если у монеты максимум 50x → ставим 50x
- Если у монеты максимум 25x → ставим 25x
- Если у монеты максимум 20x → ставим 20x
- Если не указано → по умолчанию 50x

**Пример:**
```python
leverage = config.get_leverage_for_symbol("BTCUSDT")  # → 50
leverage = config.get_leverage_for_symbol("SOLUSDT")  # → 25
leverage = config.get_leverage_for_symbol("XRPUSDT")  # → 20
```

### 3. Риск 2% на позицию (для stop loss)

**Параметр:** `RISK_PER_POSITION=2.0`

**Использование:**
- Риск 2% используется только для расчета **stop loss**
- НЕ влияет на размер позиции (размер фиксированный)

**Пример:**
```
Цена входа: 50,000 USDT
Риск: 2%
Stop loss distance: 50,000 * 0.02 = 1,000 USDT
Stop loss для LONG: 50,000 - 1,000 = 49,000 USDT
Stop loss для SHORT: 50,000 + 1,000 = 51,000 USDT
```

### 4. Максимум 10 позиций одновременно

**Параметр:** `MAX_POSITIONS=10`

**Логика:**
- Бот может открыть до 10 позиций по разным монетам
- Каждая позиция независима
- Cooldown работает отдельно для каждой монеты

## 📊 Полный пример

### Сценарий: Баланс 100 USDT

**Настройки:**
```env
FIXED_POSITION_SIZE=1.0
RISK_PER_POSITION=2.0
MAX_POSITIONS=10
LEVERAGE_CONFIG=BTCUSDT:50,ETHUSDT:50,SOLUSDT:25
```

**Открытие позиции BTCUSDT:**
```
1. Баланс: 100 USDT
2. Множитель: 100 / 100 = 1
3. Размер позиции: 1 USDT * 1 = 1 USDT
4. Цена BTC: 50,000 USDT
5. Количество BTC: 1 / 50,000 = 0.00002 BTC
6. Плечо: 50x (из конфига)
7. Маржа: 1 / 50 = 0.02 USDT
8. Нотационная стоимость: 0.00002 * 50,000 = 1 USDT
9. Stop loss: 50,000 * 0.98 = 49,000 USDT (2% риск)
```

**Открытие позиции SOLUSDT:**
```
1. Баланс: 100 USDT (тот же)
2. Множитель: 1
3. Размер позиции: 1 USDT
4. Цена SOL: 150 USDT
5. Количество SOL: 1 / 150 = 0.00667 SOL
6. Плечо: 25x (из конфига)
7. Маржа: 1 / 25 = 0.04 USDT
8. Stop loss: 150 * 0.98 = 147 USDT
```

**Итого при 10 позициях:**
```
Позиция 1: 1 USDT @ 50x = 0.02 USDT margin
Позиция 2: 1 USDT @ 50x = 0.02 USDT margin
Позиция 3: 1 USDT @ 25x = 0.04 USDT margin
...
Позиция 10: 1 USDT @ 25x = 0.04 USDT margin

Total margin used: ~0.3 USDT (очень мало!)
Total exposure: 10 USDT (10 позиций по 1 USDT)
```

### Сценарий: Баланс вырос до 250 USDT

**Открытие новой позиции:**
```
1. Баланс: 250 USDT
2. Множитель: 250 / 100 = 2
3. Размер позиции: 1 USDT * 2 = 2 USDT
4. Цена BTC: 50,000 USDT
5. Количество BTC: 2 / 50,000 = 0.00004 BTC
6. Плечо: 50x
7. Маржа: 2 / 50 = 0.04 USDT
```

## 🎯 Преимущества этой логики

### 1. Простота
- Фиксированный размер легко понять
- Не нужны сложные расчеты
- Предсказуемый риск

### 2. Масштабируемость
- Автоматически растет с балансом
- Каждые 100 USDT = +1 USDT к позиции
- Линейное масштабирование

- Учитывает ограничения биржи
- Легко настроить

### 4. Низкий риск
- Маленькие позиции (1-2 USDT)
- Stop loss всего 2%
- Максимум 10 позиций

## ⚠️ Важные замечания

### Реальный риск

При балансе 100 USDT и 10 позициях по 1 USDT:
- **Total exposure:** 10 USDT (10% баланса)
- **Риск на позицию:** 2% от 1 USDT = 0.02 USDT
- **Максимальный риск:** 10 * 0.02 = 0.2 USDT (0.2% баланса)

Это **очень консервативно**!

### Если хотите больше риска

Можно увеличить `FIXED_POSITION_SIZE`:
```env
FIXED_POSITION_SIZE=5.0      # 5 USDT вместо 1 USDT
```

Тогда при балансе 100 USDT:
- Позиция: 5 USDT
- 10 позиций: 50 USDT exposure (50% баланса)
- Риск на позицию: 2% от 5 = 0.1 USDT
- Максимальный риск: 1 USDT (1% баланса)

### Плечо и ликвидация

**Важно:** Высокое плечо = высокий риск ликвидации!

При плече 50x:
- Движение против вас на 2% = ликвидация
- Stop loss должен сработать раньше
- Но на волатильном рынке может не успеть

**Рекомендация:**
- Начните с 10x-25x
- Тестируйте на testnet
- Увеличивайте постепенно
